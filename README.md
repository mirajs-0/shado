# Shado

Behavioral machine learning system that detects unauthorized
access to mobile devices beyond the lock screen.

## What it does

Learns your unique phone behavior — app usage patterns,
timing, and interaction style. Flags anomalies when someone
else is using your device.

## Tech Stack

- Python + scikit-learn (ML model)
- React + Tailwind (dashboard)
- Firebase (data storage)
- Flask (API)
